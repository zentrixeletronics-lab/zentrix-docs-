# Registrar Entrada (/entrada)

Use o comando `/entrada` para registrar um equipamento que chegou na assistência.

---

## 📥 Como usar

Digite no Telegram:

/entrada

O bot irá solicitar as seguintes informações:

1. Nome do cliente  
2. Equipamento  
3. Problema relatado  

---

## ✅ Exemplo

Cliente: João Silva  
Equipamento: Moving Head Beam 230  
Problema: Não liga  

---

## 🔄 O que acontece depois

- Uma OS é criada automaticamente  
- Um número de OS é gerado (ex: ZTX-2026-001)  
- Uma mensagem pronta pode ser enviada ao cliente  

---

## ⚠️ Dicas importantes

- Preencha todas as informações corretamente  
- Evite abreviações confusas  
- Descreva o problema de forma clara  

---

## ❌ Erros comuns

**Erro:** deixar campos em branco  
👉 Sempre complete todas as etapas

**Erro:** descrição vaga  
👉 Exemplo ruim: "não funciona"  
👉 Exemplo bom: "não liga, sem sinal de energia"

---

## 📌 Resumo

O comando `/entrada` inicia todo o processo de atendimento.

Use com atenção para manter o controle organizado.
