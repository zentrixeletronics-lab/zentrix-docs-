# Zentrix — Sistema de Gestão via Telegram

Organize suas ordens de serviço de forma simples, rápida e sem planilhas.

O Zentrix é um assistente inteligente que funciona diretamente no Telegram, automatizando o registro de OS, geração de orçamentos e controle de status.

---

## 🚀 Como começar

1. Acesse o bot no Telegram  
2. Digite `/entrada` para registrar um equipamento  
3. Siga as instruções passo a passo  

---

## ⚙️ O que você pode fazer

- Registrar ordens de serviço
- Gerar orçamentos automaticamente
- Controlar status dos equipamentos
- Consultar histórico de clientes
